- installasi plugin matwebsite
    composer require maatwebsite/excel

- buat file import
    php artisan make:import User --model=Userapp
        * User adalah nama file import yang akan dibuat dan pastikan awalan huruf besar
        * --model=Userapp berfungsi untuk menghubungkan file import ke model Appuser atau model yang akan di gunakan

- publish plufin matwebsite
    php artisan vendor:publish --provider="Maatwebsite\Excel\ExcelServiceProvider" --tag=config
